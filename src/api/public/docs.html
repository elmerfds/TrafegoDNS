TrafegoDNS API Documentation

API for managing DNS records with TrafegoDNS

This page provides a simple overview of the TrafegoDNS API endpoints. For interactive documentation, you can use the Swagger UI:

  • Swagger UI - Interactive API documentation interface

Authentication

POST /api/v1/auth/login

Authenticate user and get access token

Request Body:

{
  "username": "admin",
  "password": "your_password"
}
POST /api/v1/auth/refresh

Refresh access token using refresh token cookie

POST /api/v1/auth/logout

Logout user and invalidate tokens

DNS Records

GET /api/v1/dns/records

Get all DNS records with optional filtering

Query Parameters:

  • page - Page number for pagination
  • limit - Records per page
  • type - Filter by record type (A, AAAA, CNAME, etc.)
  • name - Filter by record name
GET /api/v1/dns/records/:id

Get a specific DNS record by ID

POST /api/v1/dns/records

Create a new DNS record

Request Body:

{
  "type": "A",
  "name": "example.com",
  "content": "192.168.1.1",
  "ttl": 3600,
  "proxied": true
}
PUT /api/v1/dns/records/:id

Update an existing DNS record

DELETE /api/v1/dns/records/:id

Delete a DNS record

Status Information

GET /api/v1/status

Get system status information

GET /api/v1/status/health

Check API health status (public endpoint)

For more detailed information, refer to the Swagger UI.
