Getting Started

TrafegoDNS automatically manages DNS records based on Docker container labels and Traefik configuration. Here are the main API endpoints:

  • DNS Records: /api/v1/dns/records - CRUD operations for DNS records
  • Containers: /api/v1/containers - Monitor Docker containers
  • Hostnames: /api/v1/hostnames - Manage tracked hostnames
  • Orphaned Records: /api/v1/dns/orphaned - Handle cleanup of unused records
  • Authentication: /api/v1/auth - User authentication and management

Authentication

Most endpoints require authentication via JWT tokens. Use the /api/v1/auth/login endpoint to obtain a token, then include it in the Authorization: Bearer {token} header.
